DNS Zone Transfer

This is a web server vulnerability that is often left that by copying the host name files from a primary server to secondary and in between there the pentesters find a way to sneak in.

What is DNS?
Domain Name System, a protocol used to resolve domain names/hostnames to IP Addresses.

DNS Records.

DNS Interrogation:

In active recon, we use this to dip deep the ip addresses and stuff that is not publically available.

To dig deeper, here comes DNS Zone Transfer.

1. dnsenum -- dsenum zonetransfer.me
2. dig -- dig axfr @<ServerName> ZoneTransfer.me

there is this tool called Fierce that is used to scan locate non-contagious IP spave and hostnames against specified domains.

Fierce:

Brute force with fierce:
 fierce -dns zonetransfer.me

this returns that the Zone Transfer is on for the web servers
