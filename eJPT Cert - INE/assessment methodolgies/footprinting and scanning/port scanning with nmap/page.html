Port Scanning for the Open Ports:


nmap 10.10.44.4

nmap -Pn 10.10.44.4 -- not Ping

nmap -Pn -F 10.10.44.4 -- Faster

nmap -Pn -p 80 10.10.44.4 --open ports

nmap -Pn -p 80 10.10.44.4 -- more than 1 open ports

important
if it is filtered  State, it means there is windows firewall
if it is Closed, windows firewall isnt active or there are not rules

For a fast scanning of TCP ports, use all TCP ports as 65535

nmap -T4 -Pn -p- 10.10.44.4 -- -p- means all ports to scan

nmap -T4 -Pn -sS -F 10.10.44.4 -- stealthier scan, top100 ports to scan
