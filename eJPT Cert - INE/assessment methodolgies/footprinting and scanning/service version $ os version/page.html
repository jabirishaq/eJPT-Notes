Steps to discover
1. ifconfig
it reveals that eth1 ip
subnet in eth1 reveals how larger is the network

2. ip a s command
the subnet range for eth1 will be there

3. nmap -sn <eth1 ip address>/24 -- the size of network

4. if the ports are closed, usually it shows all are closed.

so we use the command

nmap -T4 -sS -p- <ip> -- to scan all ports

it will reveal all ports.

5. Now comes the service version detection scan

nmap -T4 -sS -sV -p- <ip>

this will return the services and the version too.

now note these Service+Version columns
start on the vulnerabilities and misconfigurations are done by the developers etc

6. nmap -T4 -sS -sV --version-intensity 8 -O --osscan-guess -p- <ip>

This enables to search operating system
also it guesses the os

version intesnity ranges till 9
