Network Discovery

Host Discovery Technics

it is a crucial phase to identify live hosts on a network

1. Ping Sweeps (ICMP Echo requests)
2. ARP (Address Resolution Protocol) Scanning - Local Network
3. TCP SYN PING (Half-Open Scan) if host is live it will respond with SYN-ACK
4. UDP Ping
5. TCP ACK Ping
6. SYN-ACK (Sends SYN-ACK packets)


Considerations:
ICMP Ping:
Pros: Widely supported and quick method identifying live hosts.

Cons: windows blocks it, IDS can detect it very easily


TCP SYN Ping:
Pros: stealthier than ICMP and may bypass firewalls that allows outbound connections

Cons: Some hosts may not respond to TCP SYN_ACK requests and the results can be affected by firewalls and security devices.
