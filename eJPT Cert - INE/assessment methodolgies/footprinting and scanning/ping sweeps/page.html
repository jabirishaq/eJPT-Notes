Ping Sweeps
is a network scanning technique used to discover live hosts within the specific IP Address range on a network

Basic idea is to send a series of ICMP Echo Request Messages to a range of ip addresses and observer the responses to determine which addresses are active or reachable

The windows Hosts will block by default


Ping command is also the working
fping -a -g 10.10.36.0/24 is utilizing the ICMP

-a the alive hosts
-S source IP
-A the address of the hosts
-g target list
